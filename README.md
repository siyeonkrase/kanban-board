## Kanban Board

An interactive Kanban-style task management web application designed to organize workflows through boards and draggable task cards.

### Features
- Create, rename, and delete boards
- Add, edit, and organize tasks within boards
- Drag and drop boards and task cards for flexible workflow management
- Color-coded boards for better visual organization
- Fully responsive UI for desktop and mobile usage

### Tech Stack
React, TypeScript, styled-components, HTML, CSS
